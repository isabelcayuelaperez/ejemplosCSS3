Estilos CSS: los ID

Dentro de la hoja se pueden aplicar estilos a etiquetas muy concretas a través del atributo ID.

Etiquetas <p> con distinto ID de estilo

En el fichero CSS un estilo para los ID se define con un nombre precedido por una almohadila como por ejemplo:
#menucentral { color: green; font-style: italic }

Este es el párrafo con la con el estilo #parrafo_especial
especificado en la hoja de estilos. Se aplica con el atributo id
<p id="parrafo_especial">
